Biblio-US17: Historical of proposals received

The dataset Biblio-US17 (accessible at https://doi.org/10.12795/11441/148254) contains a set of anonymized and labeled HTTP requests (selected fields) from the logs of a real-in-production web server at the library of the University of Seville during 6.5 months in 2017. Each record includes the method, URI and associated response code and size for every HTTP request received by the web server.

The dataset has been sanitized using a supervised methodology as proposed in: Díaz-Verdejo, Jesús E.; Estepa, Antonio; Estepa, Rafael; Madinabeitia, German; Muñoz-Calle, Javier, "A methodology for conducting efficient sanitization of HTTP training datasets", Future Generation Computer Systems, vol. 109, pp. 67-82, 2020. https://doi.org/10.1016/j.future.2020.03.033.

We have labeled each record as either normal or attack after a thorough semi-automated process that involved signature-based detection and detection of anomalies in the vocabulary observed in the URIs.

To allow the collaboration of researchers from the scientific community in the refinement and maintenance of the dataset, we offer the form https://javmunca2.github.io/Biblio-US17_Collaboration/form.html.
